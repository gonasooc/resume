  • 직관적인 페이지 기반의 App Router와 Server Side에서의 다양한 경험이 흥미로웠고, TypeScript를 도입하고 사용하면서 필요성을 이해하며 적극 활용해 볼 수 있었습니다.

  • 팀 단위의 다수 구성원이 참여하는 프로젝트다 보니 커뮤니케이션이 중요했습니다. Slack, Notion을 포함해 다양한 채널을 통해 효과적인 소통을 유지하고, GitHub 내에서는 Issues와 Projects, Pull requests를 활용했습니다. 이를 통해 소프트 스킬을 향상하며, 다수의 구성원들과의 협업에서 소통 비용을 최소화하는 방법을 배웠습니다.
